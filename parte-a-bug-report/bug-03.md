# Bug 03 - Layout quebra em tela pequena

## Descrição
Ao acessar o sistema em telas menores (como celular), os elementos da página ficam desorganizados e ultrapassam os limites da tela.

## Passos para reproduzir
1. Abra o site
2. Pressione F12 para abrir o modo responsivo (ou acesse pelo celular)
3. Reduza a largura da tela

## Resultado esperado
O layout deveria se adaptar corretamente a diferentes tamanhos de tela.

## Resultado obtido
Os elementos ficam sobrepostos e parte do conteúdo fica fora da tela.

## Severidade
Baixa

## Ambiente
Google Chrome (modo responsivo), Android
